# 电磁场与电磁波

老师：魏兴昌

关于老师：老师很捞！上课会写很多板书，内容可能会有一点枯燥（但是这是学科的原因不是老师的原因），会标出重点、讲解易考点，课上还有练习巩固。wxc老师的小测是助教负责，小测完会带大家过一遍答案再交上去，可能平时分会稍微好看一点。wxc老师期末还多加了一个专题研究，但专题研究的题目不难，假如想水的话很容易水过去。总之非常推荐！

关于考试：上课内容会很难听懂，但其实作业、小测、考试的难度个人感觉不大，掌握方法+拟合历年卷就可以。第二章传输线理论一定要看懂，之后的很多章节知识最后都可以用传输线等效。

关于场波实验：努力与回报不成正比的课，理论部分昏昏欲睡，实验部分兵荒马乱。建议报告不要花太多时间能水就水。
